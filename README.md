# Nclex-Keys-International

## VSCode Codex panel stuck loading (quick fix)

If the Codex tab keeps spinning for a long time, check these in order:

1. **Confirm you are signed in** to the extension account in VSCode.
2. Reload VSCode window (`Ctrl+Shift+P` → `Developer: Reload Window`).
3. Sign out/in from the extension once.
4. Update VSCode and the extension to latest versions.
5. Disable VPN/proxy/firewall temporarily, or allow VSCode extension network access.
6. Open `Help` → `Toggle Developer Tools` and check for extension errors.

If it still hangs, reinstall the extension and restart your machine.
